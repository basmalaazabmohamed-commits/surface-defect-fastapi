System Workflow

Intelligent defect detection pipeline from image capture to analysis

1

Image Capture

Upload or capture steel surface images via drag-drop interface. Accepts JPG, PNG formats with automatic quality validation.

2

Preprocessing

CLAHE enhancement, RGB conversion, and normalization to 224x224. Optimizes contrast for accurate feature extraction.

3

AI Analysis

EfficientNet-B0 CNN processes the image through 16 convolutional layers, extracting hierarchical visual features.

4

Classification

Softmax classification across 6 defect categories with confidence scores. Results displayed with probability visualization.

AI Model Details

EfficientNet-B0 deep convolutional neural network

🏗️

Architecture

  • Base Model: EfficientNet-B0
  • Input Size: 224 × 224 × 3
  • Parameters: ~5.3 Million
  • Depth: 16 Convolutional Blocks
🎓

Training

  • Dataset: NEU Steel Surface
  • Samples: 1,800 Images
  • Augmentation: Rotation, Flip, Zoom
  • Optimizer: Adam (LR: 0.001)
🎯

Performance

  • Accuracy: 99.2%
  • Precision: 98.8%
  • Recall: 99.0%
  • F1-Score: 98.9%
